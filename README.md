# Chess Simulation 

A chess console application built in Python for practice and learning purposes. Developed over the course of a month starting from April 2025. The application displays a chess board in the console, which updates
when pieces gets moved. 

## Features
- Proper piece movements according to the rules of chess
- Situational rules such as castling, en passant and promotion
- Piece capture and score tracking
- Checkmate function

## Installation
1. Clone repo and navigate to the project page
```
git clone https://github.com/NevVaek/Chess-Simulation.git
cd Chess-Simulation
```
2. Run Main.py
```
py Main.py
```
